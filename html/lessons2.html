Lesson 2

If Statements, while loops and for loops

If Statements

An if statement is a programming conditional statement that, if proved true, performs a function or displays information. If the condition is false then the code will stop, unless provided with an “esle statement” then you can make the code perform another action. An if statement also allows conditional flow, an example of how an if statement works is shown below

Syntax

if (condition) {
block of code to be executed if the condition is true
}
[imgage]

Example:

name = “bob”; if (name = “bob”){ print (“the condition is true”) else: print(“the condition is false”) }

While loops

In most computer programming languages, a while loop is a control flow statement that allows code to be executed repeatedly based on a given Boolean condition. The while loop can be thought of as a repeating if statement. An example of how a while loop works can be shown below.

Syntax

While (condition){ increment; block of code will execute for as long as the condition is met
}
[imgage]

Example:

while a = true{
print(“hello world”)
}
output: hello world, hello world, hello world, hello world, hello world, hello world…… [loops infinitely]

For Loop

A for loop is an iteration type of loop. After the body of the 'for' loop executes, the flow of control jumps back up to the increment statement. If it is true, the loop executes and the process repeats itself. After the condition becomes false, the 'for' loop terminates. Example of how for loop works.

Syntax

for (init; condition; increment) ...
}
[imgage]

Example:

for(int i=10; i>1; i--){
Print("The value of i is: "+i);
}
output of i:
10, 9, 8, 7, 6, 5, 4, 3, 2
