Understanding Integers

Integers are variables that can be used to store whole numbers. Integers can be used for math and calculations.

For Blockly

Integers store whole numbers, they can be positive, negative or zero.
Examples of integers: -5, 8, 0, 2560
Example in java: int value = “5”;
Step 1: go into the text category and grab the print block and place inside of the work area
Step 2: go to the math category and then grab the ‘a number block’ and place it beside the print block
step 3: then type 5 inside of the a number block
Step 4: click the green run button to test your code

For Scratch

Step 1: First click on the Data tab in scratch and then click “make a variable”
Step 2: Give your Variable a meaningful name like myInteger and click ok. Make sure you have “for all sprites” clicked
Step 3: Drag and drop the “set Variable to” block onto the editor
Step 4: Then click on the Events tab and drag and drop the “when clicked” block with the green flag onto the editor
Step 5: Place the “set Variable to” block under the “when clicked” so that they lock together
Step 6: In the “set Variable to” block where there is a number 0 click in that space and type a number you wish to display like 5
Step 7: Click on the Looks tab and drag and drop the “say” block under the “set Variable to” block
Step 8: Navigate back to the data tab and click on the circle with the name myInteger and drag and drop it inside the Say block
Step 9: click the Green flag at the top of the editor
