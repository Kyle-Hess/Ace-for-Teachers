Booleans

The Boolean expression has two data types are true or false
example in java: bool = true
System.out.println(bool)
Step 1: grab the print block and place inside of the work area
Step 2: grab the true or false block and place it beside the print block
Step 3: change the true or false block to true

for scratch


Step 1: First click on the Data tab in scratch and then click “make a variable”
Step 2: Give your Variable a meaningful name like myBoolean and click ok. Make sure you have “for all sprites” clicked
Step 3: Drag and drop the “set Variable to” block onto the editor
Step 4: Then click on the Events tab and drag and drop the “when clicked” block with the green flag onto the editor
Step 5: Place the “set Variable to” block under the “when clicked” so that they lock together
Step 6: In the “set Variable to” block where there is a number 0 click in that space and type either True or False
Step 7: Click on the Looks tab and drag and drop the “say” block under the “set Variable to” block
Step 8: Navigate back to the data tab and click on the circle with the name myBoolean and drag and drop it inside the Say block
Step 9: click the Green flag at the top of the editor
