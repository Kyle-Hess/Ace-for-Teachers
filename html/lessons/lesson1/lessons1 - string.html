Strings

Strings can hold sentences, names, characters, letters, numbers and special characters.
For example, in Java a name variable: String name = “John”;
Step 1 grab the print block and place inside of work area
Step 2 place the “” block beside the print block
step 3 Type John inside the “” block

for scratch

Step 1: First click on the Data tab in scratch and then click “make a variable”
Step 2: Give your Variable a meaningful name like myName and click ok. Make sure you have “for all sprites” clicked
Step 3: Drag and drop the “set Variable to” block onto the editor
Step 4: Then click on the Events tab and drag and drop the “when clicked” block with the green flag onto the editor
Step 5: Place the “set Variable to” block under the “when clicked” so that they lock together
Step 6: In the “set Variable to” block where there is a number 0 click in that space and type a name you wish to display like John
Step 7: Click on the Looks tab and drag and drop the “say” block under the “set Variable to” block
Step 8: Navigate back to the data tab and click on the circle with the name myName and drag and drop it inside the Say block
Step 9: click the Green flag at the top of the editor
